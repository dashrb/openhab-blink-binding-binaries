# openhab-blink-binding-binaries
A repo containing the released binaries from openhab-addons/bundles/org.openhab.binding.blink

The source code is in this repo: https://github.com/dashrb/openhab-addons/tree/blink2026/bundles/org.openhab.binding.blink


Tip for self: the raw download URL for a jar file listed above can be displayed via these steps:
1. right click on jar file above -> Copy Link to clipboard
2. `curl -s -I PASTE_URL | grep x-raw-download`
